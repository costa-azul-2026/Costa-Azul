# Descrição do projeto

O Costa Azul é um projeto  que consiste unificar as três princiapis entidades que compõem o ecosistema de um cruzeiro em uma plataforma digital conjuta e correlacionada, solucionando suas principais necessidades e a aprimorando suas experiências. 

O sistema se divide em 2 plataformas visando seus atores
- Aplicativo Mobile, voltado para os passageiros;
- Aplicativo Desktop, dividido em dois módulos:
    - Tripulação;
    - Companhia.

### Como funciona
Cada ator tem acesso especificos com funcionalidades direcionadas de acordo com sua posição no sistema, por exemplo um passageiro tem disponivel a programação completa do cruzeiro que ele pode usufruir. Enquanto os tripulantes alimnetam o sistema e organizam as demandas que urgem dentro do ecossistema, automaizando funções. As companhias se beneficiam dos indices coletados e gerenciamento como todo de diversos barcos disponivéis em seus respectivos nomes. 
Portanto, o Costa Azul busca reunir todo esse ecossistema em uma única plataforma, permitindo que cada parte se comunique de forma integrada e tornando a gestão do cruzeiro mais eficiente para todos.
