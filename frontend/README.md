# Estrutura do Projeto

O projeto consiste em duas aplicações:
- Aplicação mobile para os passageiros
- Aplicação/web  desktop para os tripulantes e companhias

### Porque flutter?
Flutter é orientada a aplicativos para celular, sendo assim nosso principal foco: o aplicativo para os clientes do cruzeiro ser a vitrine para uma boa experiência e profissional, logo flutter foi a linguagem escolhida para ser utlizada no front end.
